# Anant GPT

Anant GPT is a simple AI assistant built with Codex and ChatGPT assistance. It is designed to provide helpful interactions without requiring API keys or cloud-based AI services.

## Features

* No API key required
* Lightweight and easy to use
* Assistant-like interaction system
* Option-based conversation flow
* Fast and responsive
* Beginner-friendly design

## How It Works

Anant GPT uses predefined conversation logic and selectable response options to assist users. By choosing from the available options, users can navigate conversations and receive guidance on various tasks.

Since the project does not currently use a Large Language Model (LLM), it is not capable of advanced reasoning, understanding, or generating highly intelligent responses like modern AI chatbots.

## Limitations

* Not powered by an LLM
* Limited conversational understanding
* Responses depend on predefined logic
* No real-time internet access
* Cannot learn from conversations

## Purpose

The goal of Anant GPT is to demonstrate how an assistant-like application can be created without expensive AI infrastructure, API subscriptions, or cloud services.

## Future Roadmap

This project is still in its early stages. In the future, when funding and resources become available, I plan to integrate LLM APIs into Anant GPT.

Potential upgrades include:

* Integration with modern LLMs
* Natural language understanding
* Context-aware conversations
* Improved reasoning capabilities
* Personalized assistance
* Real-time information access

With these enhancements, Anant GPT can evolve from a rule-based AI assistant into a truly intelligent LLM-powered assistant capable of more natural and meaningful interactions.

## Tech Stack

* Codex-assisted development
* Rule-based conversation system
* Lightweight architecture

---

**Today, Anant GPT is a simple assistant designed to help users through structured interactions. Tomorrow, it aims to become a fully intelligent LLM-powered assistant.**
